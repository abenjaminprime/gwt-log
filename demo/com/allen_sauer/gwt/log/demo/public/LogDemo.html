gwt-log - Runtime logging for your Google Web Toolkit projects.
Features:
  • Logging code is compiled out and introduces zero-overhead* due to GWT 1.5 compiler dead code elimination when log_level=OFF
  • Six out of the box log levels (DEBUG, INFO, WARN, ERROR, FATAL, OFF), with the flexibility to add your own log levels
  • Log level can be controlle via log_level URL parameter, GWT meta tag in your HTML, or in your module *.gwt.xml file
  • Wide variety of context sensitive log destinations: popup DIV, Firebug, console.log(), GWT.log(), System.err()/System.out()
  • Ability to independently control compile time and runtime log levels

* Unless your logging parameters are determined to have side effects
